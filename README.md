# Human Pose Estimation using MediaPipe, MoveNet and YOLOv8 Pose

## Overview

This project implements human pose estimation using three different models:

* MediaPipe Pose
* MoveNet
* YOLOv8 Pose

The models are tested on:

* Static Images
* Real-Time Webcam Streams

## Technologies Used

* Python
* OpenCV
* TensorFlow
* TensorFlow Hub
* MediaPipe
* YOLOv8
* NumPy

## Implementations

### MediaPipe

* Static Image Pose Detection
* Real-Time Pose Detection

### MoveNet

* Static Image Pose Detection
* Real-Time Pose Detection

### YOLOv8 Pose

* Static Image Pose Detection
* Real-Time Pose Detection

## Features

* Human body keypoint detection
* Real-time webcam pose estimation
* Multi-model comparison
* Computer Vision based pose tracking

## Models Comparison

| Model       | Keypoints | Speed     | Multi-Person Support |
| ----------- | --------- | --------- | -------------------- |
| MediaPipe   | 33        | Fast      | Limited              |
| MoveNet     | 17        | Very Fast | Single Person        |
| YOLOv8 Pose | 17        | Medium    | Yes                  |
